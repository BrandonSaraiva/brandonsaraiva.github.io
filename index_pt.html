Portfolio

Projetos nos quais apliquei minhas habilidades como cientista de dados. Cada um deles representou uma oportunidade de aprendizagem e aplicação prática de conceitos-chave, resultando em soluções funcionais e inovadoras.

Bot Telegram Lotofácil

Um bot Python que usa dados históricos de sorteios de loteria para analisar os números sorteados com maiores frequências em uma determinada data ao longo dos anos. Com base nesta análise, o bot gera uma previsão dos números com maior probabilidade de serem sorteados no dia atual com base no passado.

Bot Telegram Home Pc

Aplicativo Python para automatizar tarefas diárias. Com comandos simples, posso receber faturas, enviar arquivos para o Google Drive e acessar informações úteis, como previsão do tempo e últimos e-mails. Além disso, implementei recursos para controlar remotamente o PC e reproduzir vídeos no YouTube.

Gift Cards Website

Foi desenvolvido um website para simplificar a gestão de cartões presente numa loja. Esta plataforma gera automaticamente cartões-presente contendo códigos exclusivos e datas de validade, fornecendo uma solução integrada com um banco de dados privado. O aplicativo foi desenvolvido para oferecer facilidade de uso, bastando executar o aplicativo para estabelecer e manter o controle eficiente dos vales-presente da loja. Esta ferramenta visa otimizar a administração desses recursos.

Projeto Eleições Gerais - IESB

Desenvolvemos um projeto de análise descritiva dos dados das Eleições Gerais Ordinárias no Brasil em 2022, com foco no Distrito Federal. Usamos PostgreSQL como banco de dados e Python/SAS para análise. Identificamos padrões, tendências e percepções relevantes nos resultados eleitorais.

Pomodoro Mobile App

Aplicativo Pomodoro personalizado para uso pessoal que criei depois de ficar frustrado com as opções disponíveis na Play Store. A interface é propositalmente única e simples. O aplicativo está disponível para download no formato .pkg para uso no seu celular. Transformar o código Python em um executável .pkg foi um desafio, mas depois de aproximadamente 9 horas de trabalho, consegui fazê-lo funcionar.

Projeto de Internações e Óbitos - IESB

Aplicativo que investiga a relação entre internações, tempo de internação e mortalidade. Foi utilizado dados públicos do Ministério da Saúde para analisar o número de internações, óbitos e o tempo médio de internação por procedimento em cada município brasileiro.

PI3 webscraping project - IESB

Automação de processos de análise de dados. Utilizei diversas bibliotecas, como pyautogui, Selenium e pandas. O bot gera slides com base nas informações fornecidas e cria gráficos a partir de um banco de dados baixado. O projeto inclui interações sonoras para guiar a apresentação.

Envio automatizado de mensagens

Este projeto teve como objetivo automatizar o envio de mensagens em horários agendados e permitir o agendamento para qualquer dia desejado. Além disso, também coleta moedas diariamente na plataforma Shopee. A proposta é oferecer uma solução simples e eficiente para essas tarefas específicas. Com este bot é possível automatizar o processo de envio de mensagens e garantir que as moedas sejam recolhidas diariamente, sem necessidade de intervenção manual.

Indexador de documentos para Elasticsearch

Este é um projeto que permite indexar documentos no Elasticsearch com base em URLs fornecidas em arquivos JSON. Ele fornece flexibilidade para selecionar uma coluna específica dos dados JSON e coletar o conteúdo da URL se essa coluna contiver um link HTTP para algum documento.

Random Thread Posts Scraper

Este script Python permite recuperar postagens aleatórias do perfil de qualquer usuário na plataforma de mídia social emergente, Threads. Como Threads atualmente não possui uma API pública para desenvolvedores, este projeto usa web scraping com Selenium para buscar o conteúdo HTML de perfis de usuários e extrair suas postagens.
